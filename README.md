# News-X
News-X is a web App that you can check the latest news from it.
## Getting started
Project was created with:
* ReactJs
* NodeJs
* Express
* MongoDB

## Try News-X Localy
```
$ git clone https://github.com/AhmedAdelFahim/News--X.git
```
After you clone the project follow the next steps:
```
$ cd News--X/server
$ npm install
$ check .env.example
$ npm start
$ cd ../client
$ npm install
$ npm start
Enjoy:)
```
